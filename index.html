Ingeniería Industrial

Power BI

Power BI es una plataforma de Microsoft que ayuda a recopilar, administrar, y analizar datos de múltiples fuentes, permite tomar decisiones basadas en datos para procesos industriales. Tiene funcionalidades de inteligencia artificial y una herramienta de creación de informes fácil de usar.

IR A WEB

Looker Studio

Looker Studio es una herramienta de inteligencia que permite a los usuarios visualizar y analizar datos, crear informes y compartirlos con otros. Es una herramienta gratuita para los creadores y lectores de informes, pero también existe una versión superior llamada Looker Studio Pro.

IR A WEB

Any Logic

Any Logic es una herramienta de simulación industrial que permite modelar sistemas de fabricación, transporte y logística. permite a los analistas, ingenieros y gerentes obtener conocimientos más profundos y optimizar sistemas y procesos complejos.

IR A WEB

Ingeniería en Sistemas

GitHub Copilot

Copilot es un asistente de códificación que utiliza IA para sugerir cógido y funciones en tiempo real. Te ayudará a escribir código más rápido y ser más eficiente. Cuenta con uan versión gratuita y una premiun y está disponible como puglin de VSCode.

IR A WEB

PyTorch

PyTorch es una biblioteca de código abierto basado en software que se utiliza para crear redes neuronales, combinando la biblioteca de machine learning (ML) de Torch con una API de alto nivel basada en Python.

IR A WEB

WEKA

WEKA almacenar, procesar y gestionar datos de forma sencilla y sostenible prácticamente en cualquier lugar con la simplicidad de la nube y el rendimiento local. Una plataforma de datos nativa de IA.

IR A WEB

Google Sites

Sites es una aplicación gratuita de Google que permite crear sitios web para uso personal o profesional. Es una herramienta que facilita la creación de páginas web, ya que no requiere conocimientos de programación y se puede usar de manera similar a un editor de documentos.

Ingeniería Mecánica

OpenModelica

OpenModelica es una plataforma gratuita que permite modelar, simular, optimizar y analizar sistemas mecánicos y dinámicos complejos. Se basa en el lenguaje de modelado Modelica, que es un lenguaje orientado a objetos.

IR A WEB

SIMSCALE

Simscale es una plataforma en línea gratuita que sirve para simulación de fluidos, análisis estructural y térmico. Ideal para proyectos mecánicos

IR A WEB

FreeCad

FreeCAD es un software de diseño asistido por computadora (CAD) en 3D donde puedes modelar objetos reales de cualquier tamaño: Diseñar productos, Ingeniería mecánica o Arquitectura.

IR A WEB

CalculiX

CalculiX es un programa de análisis de elementos finitos de código abierto y gratuito que sirve para resolver problemas de campo. Se utiliza para construir, calcular y post-procesar modelos de elementos finitos.

IR A WEB

Ingeniería Ambiental

Google Earth Engine

Google Earth Engine es una plataforma a escala planetaria para datos y análisis de las ciencias de la Tierra, combina un catálogo de imágenes satelitales y conjuntos de datos geoespaciales de varios petabytes con capacidades de análisis a escala planetaria.

IR A WEB

openLCA

OpenLCA es un software gratuito y de código abierto para realizar análisis de ciclo de vida dando información muy detallada sobre los resultados, útil para evaluar el impacto ambiental de productos y procesos.

IR A WEB

Ingeniería de Alimentos

FoodAI

FoodAI es una plataforma gratuita para analizar tendencias alimentarias, optimizar recetas y mejorar procesos utilizando algoritmos de IA.

IR A WEB

Orange Data Mining

Orange data mining es un software gratuito y de código abierto que permite analizar datos relacionados con calidad de alimentos, procesos de producción y predicción de demanda.

IR A WEB

Bio Numbers

Bio Numbers es una base de datos gratuita que contiene datos cuantitativos sobre propiedades biológicas y procesos relacionados con alimentos y nutrición.

Ingeniería Agronómica

PlantSnap

PlantSnap es una aplicación basada en AI para ayudar a identificar especies vegetales y de hongos. Permite reconocer la mayor parte de las plantas y árboles, guardar las favoritas y crear colecciones. Además se pueden realizar consultas sobre la clasificación taxonómica y distribución de todas las especies que contiene la aplicación a nivel global.

IR A WEB

Agremo

Agremo es una plataforma que utiliza IA para analizar imágenes de cultivos tomadas por drones y satélites. Ofrece análisis de salud del cultivo, estimación de rendimiento y densidad de siembra.

IR A WEB

AgriMetSoft

Agrimetsoft es un software gratuito que te ayuda a realizar análisis climáticos y agrícolas utilizando IA para predecir rendimientos y optimizar prácticas agrícolas.
